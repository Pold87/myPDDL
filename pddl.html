What is PDDL? A short PDDL tutorial

First of all, PDDL is a language for planning tasks and stands for Planning Domain Definition Language. Planning is about what to do and in which order for a given problem. In fact, PDDL is thestandard language for Artificial Intelligence (AI) problems. There are several versions of PDDL, i.e. the language changes over time and new features are added. If this sounds cryptic to you, you'll get a better idea if you'll continue to read this tutorial. I think it's best to have an example.
