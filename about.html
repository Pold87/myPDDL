About

I am a cognitive science student at the University of Tuebingen. This website was created for my bachelor thesis that is about AI planning using the Planning Domain Definition Language PDDL. Searching for an efficient way of writing PDDL domains and problems, I decided to create a PDDL package for the awesomeSublime Text Editor.

If you have any questions, issues or ideas, please feel free to contact me. My email address is:
volker.strobel87@gmail.com

Related work:

  • PDDL Studio by Miroslav Chomut and Tomas Plch - an IDE for developing PDDL documents
  • PDDL Mode by Surendra K Singhi - an extension for GNU Emacs that supports PDDL 2.2
